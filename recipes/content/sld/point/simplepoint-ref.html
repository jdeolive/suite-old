Simple Point Documentation

This example specifies points be styled as red circles with a diameter of 6 pixels.
There is one <Rule> in one <FeatureTypeStyle> for this SLD, which is the simplest possible situation. (All subsequent examples will contain one <Rule> and one <FeatureTypeStyle> unless otherwise specified.) Styling points is accomplished via the <PointSymbolizer> (lines 8-18). Line 11 specifies the shape of the symbol to be a circle, with line 13 determining the fill color to be red (#FF0000). Line 11 sets the size (diameter) of the graphic to be 6 pixels.
