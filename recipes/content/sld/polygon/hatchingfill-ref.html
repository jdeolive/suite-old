Hatching Fill

This example fills the polygons with a hatching pattern.
In this example, there is a <GraphicFill> tag as in the Graphic fill example, but a <Mark> (lines 12-18) is used instead of an <ExternalGraphic>. Line 13 specifies a "times" symbol (an "x") be tiled throughout the polygon. Line 15 sets the color to purple (#990099), line 16 sets the width of the hatches to 1 pixel, and line 19 sets the size of the tile to 16 pixels. Because hatch tiles are always square, the <Size> sets both the width and the height.

Note: This example leverages an SLD extension in GeoServer. Hatching is not part of the standard SLD 1.0 specification.
