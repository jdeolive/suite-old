Getting Started
===============

Welcome to the OpenGeo Suite!  The OpenGeo Suite is a full geospatial software stack that allows you to allows you to compose, style, and publish maps with ease.  This **Getting Started** guide will guide you through the basic functionality of the OpenGeo Suite, including a simple workflow for creating a map.  More detailed information is available in the other included documentation projects.
