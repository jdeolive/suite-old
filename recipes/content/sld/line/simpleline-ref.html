Simple Line Documentation

This example specifies lines be colored black with a thickness of 3 pixels.
There is one <Rule> in one <FeatureTypeStyle> for this SLD, which is the simplest possible situation. (All subsequent examples will contain one <Rule> and one <FeatureTypeStyle> unless otherwise specified.) Styling lines is accomplished via the <LineSymbolizer> (lines 8-13). Line 10 specifies the color of the line to be black (#000000), while line 11 specifies the width of the lines to be 3 pixels.
