Railroad Documentation

This example uses hatching to create a railroad style. Both the line and the hatches are black, with a 2 pixel thickness for the main line and a 1 pixel width for the perpendicular hatches.
In this example, there are two rules, each containing a <LineSymbolizer>. (Each <LineSymbolizer> must exist in its own rule.) The first rule, on lines 7-13, draws a standard line, with line 10 drawing the lines as dark gray (#333333) and line 11 setting the width of the lines to be 2 pixels.

The hatching is invoked in the second rule, on lines 15-32. Line 21 specifies that the rule draw a vertical line hatch (``shape://vertline``) perpendicular to the line geometry. Lines 23-24 set the hatch color to dark gray (#333333) and width to 1 pixel. Finally, line 27 specifies both the length of the hatch and the distance between each hatch to both be 12 pixels.

Note: This example leverages an SLD extension in GeoServer. Hatching is not part of the standard SLD 1.0 specification.
