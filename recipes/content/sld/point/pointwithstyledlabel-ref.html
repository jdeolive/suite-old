Point with Styled Label Documentation

This example improves the label style from the Point with default label example by centering the label above the point and providing a different font name and size.
In this example, lines 8-18 are identical to the Simple point example above. The <TextSymbolizer> on lines 19-44 contains many more details about the label styling than the previous example, Point with default label. Lines 20-22 once again specify the "name" attribute as text to display. Lines 23-28 set the font information: line 24 sets the font family to be "Arial", line 25 sets the font size to 12, line 26 sets the font style to "normal" (as opposed to "italic" or "oblique"), and line 27 sets the font weight to "bold" (as opposed to "normal"). Lines 29-40 (<LabelPlacement>) determine the placement of the label relative to the point. The <AnchorPoint> (lines 31-34) sets the point of intersection between the label and point, which here (line 32-33) sets the point to be centered (0.5) horizontally axis and bottom aligned (0.0) vertically with the label. There is also <Displacement> (lines 35-38), which sets the offset of the label relative to the line, which in this case is 0 pixels horizontally (line 36) and 5 pixels vertically (line 37). Finally, line 42 sets the font color of the label to black (#000000). The result is a centered bold label placed slightly above each point.
