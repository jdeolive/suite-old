Components

The OpenGeo Suite is a fully-integrated open source geospatial platform for serving maps and data through the web. Explore and configure each component below:

PostGIS

  • Manage
  • Import shapefiles
  • Documentation

PostGIS “spatially enables” the PostgreSQL open source relational database. The database can then be used to store and query spatial data (points, lines and polygons).

GeoServer

  • Configure
  • Import data
  • Logs
  • Documentation

GeoServer allows users to share and edit geospatial data. Designed for interoperability, it publishes data from any major spatial data source using open standards.

GeoWebCache

  • Configure
  • Documentation

GeoWebCache accelerates the delivery of images and data by dividing GeoServer output into tiles and caching for fast delivery.

GeoEditor

  • Launch
  • Documentation

GeoEditor is a 100% web based application for managing data entry and manipulation.

Styler

  • Launch
  • Documentation

Styler brings the power of desktop GIS styling to the web, providing an easy-to-use graphical interface to customize the look of your maps.

GeoExplorer

  • Launch
  • Documentation

GeoExplorer is a JavaScript user interface framework for building browser-based geospatial applications with the look, feel, and functionality of desktop GIS clients.
