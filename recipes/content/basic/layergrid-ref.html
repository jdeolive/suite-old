Layer Grid Documentation

The MapPanel exposes a layers property as an Ext.data.Store. This store automatically updates as layers are added, removed, changed, or reordered. Using the Ext.grid.GridPanel, you can populate a list of layers in a map.
This recipe uses the following classes:
  • GeoExt.MapPanel
  • Ext.grid.GridPanel
  • OpenLayers.Layer.WMS
