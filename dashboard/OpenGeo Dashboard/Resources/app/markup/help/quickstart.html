Getting Started

This page contains a summary of how to get your data published on the web using the OpenGeo Suite. For more information, please see the OpenGeo Suite User Manual.

Step 1: Load your data

Use pgShapeLoader to convert shapefiles to table in your PostGIS database. View your data with pgAdmin.

  • If you are looking for data to get started, the U.S. Census Bureau has free shapefiles available for download.

Step 2: Serve your data

Use the Layer Importer to load your database tables or shapefiles into GeoServer. Each database table or shapefile will become an individual map layer.

Step 3: Edit your data

Use GeoEditor to graphically edit any data served through GeoServer.

Step 4: Style your map layers

Use Styler to design the look and feel for each map layer.

Step 5: Create your map

Use GeoExplorer to create a map composed of one or more map layers. Adjust zoom and pan to customize your map's view. Then publish your map on the web via a URL bookmark or via embedded HTML.

Advanced: Make your own mapping application

Read the Recipe Book to learn how to make your own customized mapping applications using the OpenGeo Suite.
