Polygon with Default Label

This example shows a text label on the polygon. In the absence of any other customization, this is how a label will be displayed.
In this example there is a <PolygonSymbolizer> and a <TextSymbolizer>. Lines 8-16 comprise the <PolygonSymbolizer>. The fill of the polygon is set on line 10 to a light green (#40FF40) while the stroke of the polygon is set on lines 13-14 to white (#FFFFFF) with a thickness of 2 pixels. The label is set in the <TextSymbolizer> on lines 17-21, with line 19 determining what text to display, in this case the value of the "name" attribute. All other details about the label are set to the renderer default, which here is Times New Roman font, font color black, and font size of 10 pixels.
