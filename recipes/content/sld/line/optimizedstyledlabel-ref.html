Optimized Styled Label

This example improves the style of optimally-placed labels
This example is similar to the Optimized label placement example. The only difference is in the font information, which is contained in lines 20-25. Line 21 sets the font family to be "Arial", line 22 sets the font size to 10, line 23 sets the font style to "normal" (as opposed to "italic" or "oblique"), and line 24 sets the font weight to "bold" (as opposed to "normal").

Note: This example uses options that are specific to GeoServer and are not part of the SLD 1.0 specification.
