Line with Default Label

This example shows a text label on the simple line. This is how a label will be displayed in the absence of any other customization.
In this example, there is one rule with a <LineSymbolizer> and a <TextSymbolizer>. The <LineSymbolizer> (lines 8-12) draws red lines (#FF0000). Since no width is specified, the default is set to 1 pixel. The <TextSymbolizer> (lines 13-18) determines the labeling of the lines. Lines 14-16 specify that the text of the label will be determined by the value of the "name" attribute for each line. Line 18 sets the text color to black. All other details about the label are set to the renderer default, which here is Times New Roman font, font color black, and font size of 10 pixels.
